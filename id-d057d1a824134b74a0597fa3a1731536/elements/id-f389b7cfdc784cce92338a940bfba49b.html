Fleet Operator ()
The Fleet Operator is responsible for the planning and execution of the fleet operations. The planning includes resource allocations, time schedule planning, etc. and is done across all the available demands from different Transport Service Providers. The operations are transport between location that must be carried out according to rules and regulations.
