Dynamic guidance for disrupted agents ()
Real-time monitoring of incoming trucks/throughput of user to enable optimal ETA in case of disruption. Polycentric approach will be deployed, by creating zones of influence managed by BDI (Belief-Desire-Intention) Infrastructure agents. Infrastructure agents will have information for traffic inside their zones and partial communication with other Infrastructure agents. Vehicle traffic inside the zones is addressed, but not orchestrated, only guidance to disrupted agents A traffic simulator software MATSim/Aimsun will be used to model the agents. Clustering of regions will be conducted, using the Snake algorithm to divide the region in homogeneous zones. Use in LL and sim: Used in the Simulation. Dynamic guidance will determine actual performance of the system and robustness.
