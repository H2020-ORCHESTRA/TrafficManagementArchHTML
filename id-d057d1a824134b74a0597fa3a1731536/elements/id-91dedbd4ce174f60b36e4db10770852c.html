Job Generation for CAVs ()
Bundling algorithm designed to create bundles of incoming trucks that can be serviced in parallel. Each bundle returns total turnaround and waiting time of each truck in the bundle, optimal order of pick-ups and drop-offs, as well as exit time of each truck in the port. Use in LL and sim: Used in the Simulation. Demand will be generated stochastically, but will enforce the structure and guidelines of the proposed functionality.
