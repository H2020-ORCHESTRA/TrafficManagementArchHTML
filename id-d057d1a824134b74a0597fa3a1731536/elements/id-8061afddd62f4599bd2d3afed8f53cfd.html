Request Transport Operation in Network ()
Request a new transport operation, or updates to an existing operation, in the network Can include a time range within which the transport should occour (departure and arrival time), price preferences, priority requests, capacity information, etc. Includes requested route, or at least start and end point. The request should include information about the nature of the transport, e.g. person transport, dangerous goods, goods with special needs (hot / cold / fragile). Also identifies type of transport, e.g. emergency service, road maintenance, ...
